# Scoria
...a not so lightweight language written in Kotlin/Native.

## Syntax
```rs
fn main() {
    let x = 0;
    x = x + 1;
    
    let y = 3;

    if 25 > 24 {
      : "true...";
    }
  
    if 43 <= 42 {
      y = y + 1; 
      : "this should not work!";
    }

    : y;
    : "hi"; // hi
}
```
